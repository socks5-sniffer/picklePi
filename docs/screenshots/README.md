# Screenshots Directory

This folder contains screenshots of the picklePi application for the README.

## Adding Screenshots

Before publishing to GitHub, add the following screenshots to this directory:

1. **progress-tracker.png** - Screenshot of the Progress Tracker dashboard
2. **code-view.png** - Screenshot of the code view with syntax highlighting
3. **hardware-setup.png** - Screenshot of the hardware setup instructions

## Recommended Screenshot Settings

- **Format:** PNG (for best quality) or JPG
- **Resolution:** 1920x1080 or higher for desktop views
- **File Size:** Optimize to under 500KB each for faster loading

## How to Capture

1. Run `npm run dev` to start the application
2. Navigate to each view (Progress Tracker, Project Code, Hardware Setup)
3. Use your OS screenshot tool or browser DevTools to capture
4. Save the images with the exact filenames listed above
5. Optimize using tools like [TinyPNG](https://tinypng.com/) if needed
