# Welcome to the picklePi Wiki! 🥒⚡

**picklePi** is a structured, gamified, project-based learning platform designed to teach electronics and Python programming using the Raspberry Pi.

This wiki is your central hub for understanding the architecture, curriculum, and development practices of picklePi.

## 📖 What is picklePi?

picklePi helps learners progress from their first LED blink to building a complete multi-module alarm system. It is designed with a client-first, backend-ready architecture, currently storing progress via `localStorage` but fully prepared for a relational database backend.

## ✨ Key Features

- 🎮 **Gamified Progress**: Earn unique badges as you complete each project.
- 📓 **Lab Notebook**: Structured reflection journal for every completed project.
- 📌 **Interactive Pinout**: Built-in Raspberry Pi GPIO pin map.
- 📖 **Electronics Dictionary**: Searchable glossary of Python, Pi, and electronics terms.
- 🔒 **Local Persistence**: Progress survives page refreshes via `localStorage`.
